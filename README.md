# Educational Organisation Using ServiceNow

## Overview
A web-based Educational Management System integrated with ServiceNow for managing students, teachers, admissions, and progress tracking.

## Features
- Student & Teacher Management
- Admission Workflow
- Progress Monitoring
- ServiceNow Ticketing Integration

## Tech Stack
- HTML, CSS, JavaScript
- ServiceNow Platform

## Setup
1. Clone repo
2. Open `index.html` in browser
3. Configure ServiceNow API in `script.js`
